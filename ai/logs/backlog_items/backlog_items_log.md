# 🤖 AI Usage Log – Backlog Planning for File-Flow Project

## 📌 Project Context

Development of *File-Flow*, a lightweight command-line tool designed to validate, **organise**, classify, and report on files placed within a shared team folder. The tool is intended to maintain a clean and usable shared environment by enforcing naming conventions, separating valid and invalid files, and generating logs and reports.

As part of the project, a structured backlog was required to support timely and organised delivery of all required features and deliverables.
